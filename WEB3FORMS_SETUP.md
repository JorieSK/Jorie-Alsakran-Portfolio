# إعداد Web3Forms لإرسال الرسائل (بدون Gmail مباشر)

## لماذا Web3Forms؟
- ✅ **لا يحتاج إلى ربط Gmail مباشرة**
- ✅ **سهل الإعداد** - فقط احصل على Access Key
- ✅ **مجاني** - 250 رسالة شهرياً
- ✅ **يُرسل مباشرة إلى إيميلك** - joriealsakran@gmail.com

## الخطوات السريعة:

### 1. الحصول على Access Key
- اذهب إلى [https://web3forms.com/](https://web3forms.com/)
- **مهم جداً**: أدخل إيميلك: **joriealsakran@gmail.com** (يجب أن يكون هذا الإيميل بالضبط)
- اضغط "Get Your Access Key"
- انسخ الـ Access Key الذي سيظهر لك
- **ملاحظة مهمة**: Web3Forms يرسل الإيميلات إلى الإيميل الذي استخدمته عند إنشاء Access Key

### 2. إضافة Access Key إلى المشروع
- أنشئ ملف `.env` في جذر المشروع (إذا لم يكن موجود)
- أضف السطر التالي:
  ```
  VITE_WEB3FORMS_ACCESS_KEY=your_access_key_here
  ```
- استبدل `your_access_key_here` بالـ Access Key الذي حصلت عليه

### 3. إعادة تشغيل السيرفر
- أوقف السيرفر (Ctrl+C)
- شغله مرة أخرى:
  ```bash
  npm run dev
  ```

## ملاحظات:
- ✅ **لا تحتاج إلى إعداد Gmail أو أي خدمة بريد أخرى**
- ✅ الرسائل تُرسل مباشرة إلى joriealsakran@gmail.com
- ✅ الاسم يُستخدم كعنوان الإيميل (Subject)
- ✅ البريد الإلكتروني للمستخدم يُستخدم كجهة الإرسال (From)
- ✅ الرسالة تُرسل في محتوى الإيميل

## إذا لم تقم بإعداد Web3Forms:
- سيستخدم النظام `mailto:` كبديل (يفتح Gmail فقط، لا يرسل تلقائياً)
